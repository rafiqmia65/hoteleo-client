# 🏨 Hoteleo - Hotel Booking Web Application (Client)

**Hoteleo** is a modern, responsive hotel booking web application. This client-side project is built using **React 19**, **Vite**, **Tailwind CSS 4**, and **Firebase**, offering users a seamless experience to browse rooms, book accommodations, and leave reviews.

---

## 🚀 Live Preview

👉 [Live Site](https://hoteleo.netlify.app/)

---

## 🏁 Getting Started

This is the **client-side** of the Hoteleo project, built with the latest frontend technologies for fast, responsive, and interactive experiences.

### ⚙️ Setup & Installation

#### 1️⃣ Clone the Repository

```bash
git clone https://github.com/rafiqmia65/hoteleo-client.git
cd hoteleo-client

npm install

VITE_API_KEY=your_api_key
VITE_AUTH_DOMAIN=your_auth_domain
VITE_PROJECT_ID=your_project_id
VITE_STORAGE_BUCKET=your_storage_bucket
VITE_MESSAGING_SENDER_ID=your_sender_id
VITE_APP_ID=your_app_id

npm run dev

🌟 Features
⚛️ Built with React 19 and Vite for blazing-fast performance

🎨 Tailwind CSS 4 and DaisyUI for elegant and responsive UI

🔐 Firebase Authentication (Email/Password + Google)

🗓️ Dynamic room filtering with date and price range

📷 Masonry-style photo gallery with Lightbox preview

📍 Interactive map integration using React Leaflet

📦 Secure API calls using Axios

🔔 Alerts & notifications with SweetAlert2

🗖️ Date selection using React Datepicker

🎥 Smooth animations with Framer Motion

🌐 SEO-ready using React Helmet
```
